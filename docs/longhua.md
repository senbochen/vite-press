# 龙华-深圳北站

::: tip 微博
今天 D671 比计划时间提前 18 分钟到达深圳北站，当我把列车停稳在 5 号站台一霎那 ，客室里传来乘客们惊讶的话语：乘务员小姐， 这是深圳北站吗 ？ 不是 23：38 到吗？[汗]现在才 23：20 分 其实我很想对这些乘客说：没错[噢耶]今天开的是快了点[纠结] 其实我是想赶紧下班上床睡觉 明早 6 点还要起床[泪流满面]
:::

::: tip 微博
从福建厦门(厦门北站)到广东深圳(深圳北站)的动车组列车，原来的速度是每小时 200 公里(每小时 124 英里)左右，现在从厦门到深圳的动车最快速度可以超过每小时 245 公里(每小时 152 英里)！从厦门坐动车到深圳的时间也更快了！从福州坐动车去深圳，福州到厦门速度是每小时 200 公里，过了厦门时速就能达到 245 公里！
:::

::: tip 微博
从机场到深圳北站，需要从航站楼乘地铁 11 号线转 5 号线。
不打车，更多是因为这两天总晕车，也许是胃不舒服的缘故。
中午的 5 号线，车厢宽敞有座位。于是拉着行李箱，也不至于太费劲。
对坐一对小情侣，大概二十岁左右的样。那个年龄的，瘦弱的身体，女生皮肤很白，靠着男生的肩膀，一直在睡觉。男生很自然地抱着她，偶尔调整一下姿势，让女朋友更舒适一点。
那是青春的模样，几近一无所有，都拥有着一生当中最可贵的单纯和幸福。彼此情有独衷地相爱，彼此不嫌弃，一起捱过无能为力的日子。
后来……想到后来，大多是令人感伤的后来。
我看着他们，直到发现，“下一站是翻身”。原来，这就是“翻身站”。
回福州路上了，路过惠州，车窗外，有盛开着花朵的苦楝树，有茂盛的牵牛花爬在旧墙上，有瓦房顶。
